# debut

> **Warning**  
> This project is unmaintained.

Twitch extension created for my debut stream on [Twitch](https://twitch.tv/lexikiq).
It provided viewers with some dialogue from a "hacker"
followed by a polling menu that allowed them to collectively manipulate and sabotage the stream.

## /ebs

The Extension Backend Server (EBS). Written in Kotlin using the Ktor framework.

## /frontend

The extension frontend. Written in TypeScript using the SvelteKit framework.
